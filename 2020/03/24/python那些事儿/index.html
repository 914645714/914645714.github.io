解决pip下载缓慢问题,提高下载速度(Mac)

原创:qq_41770825
最开始接触pip,是学习python爬虫的时候,下载安装包一直都是4k,极其缓慢,后来百度到

pip install -i https://pypi.tuna.tsinghua.edu.cn/simple <库名>

可以直接用国内镜像,下载速度和之前比起来,快的真的是太多了
但是一直用链接下载比较麻烦,而且每次要输很长一段代码,于是我又百度到一个更方便的操作

1.首先打开我们的Application Support文件夹

command+空格 # 聚焦搜索
输入~/Library/Application Support

2.查看所打开的文件夹有没有pip/pip.conf文件

如若没有,则新建一个pip/pip.conf文件
终端创建

cd ~/Library/Application Support #进入文件所在位置
mkdir .pip
touch pip.conf #创建pip.conf文件

然后pip/pip.conf文件就在Application Support文件夹建好了

3.编辑pip/pip.conf文件内容

[global]
index-url = https://pypi.tuna.tsinghua.edu.cn/simple/
[install]
trusted-host=pypi.tuna.tsinghua.edu.cn

4.command+v保存退出

现在我们pip install <库名> 镜像下载就ok了
