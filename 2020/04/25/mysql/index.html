Mac下连接本地数据库，出现Access denied for user ‘root‘@’localhost’ (using password: YES)

转载:IDonng__
连接本地服务器时候,我们会出现Access denied for user ‘root‘@’localhost’ (using password: YES)情况
这个意思是‘拒绝用户“root”@“localhost”的访问（使用密码：YES ’
下面是解决方案(请按步骤进行)

1.打开偏好设置->点击最下面mysql,并关闭mysql服务

2.进入终端

  • 输入

    cd /usr/local/mysql/bin/ #进入mysql主目录,点击回车

  • 输入

    sudo su #回车,获取管理员权限

  • 输入

    ./mysqld_safe –skip-grant-tables & #回车以禁止mysql验证功能，mysql会自动重启，偏好设置中的mysql状态会变成running

3.进入后会出现sh#3.2(大概会出现这样,具体字母数字记不得了)

  • 继续输入

    ./mysql #回车

  • 输入

    flush privileges; #注意后面有;

  • 输入

    ALTER user ‘root‘@’localhost’ IDENTIFIED BY ‘你的密码’ ; #注意后面有;

最后在终端输入 mysql -uroot -p 登陆来验证是否修改成功吧!
